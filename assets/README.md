# Repository Assets

This folder contains the visual documentation and interface demonstrations for the **Industrial Economic Feasibility Tool**. These images are used to provide a visual overview of the application's UI/UX and reporting capabilities.

## 📸 Included Visuals
* **`main_ui.png`**: Demonstrates the PyQt5 multi-tab interface (Upgrades, Running Costs, and Savings).
* **`graphs_tab.png`**: Displays the Matplotlib integration showing cumulative cash flow and NPV trends.
* **`numeric_results.png`**: Highlights the final calculation summary page.

## 📊 Data Export Capabilities
As demonstrated in the numeric results interface, the tool allows users to export processed data in two primary formats:

1. **PDF Export**: 
   * Generates a high-quality, formatted report using **ReportLab**.
   * Includes project recommendations, summary metrics (NPV, IRR, ROI), and professional branding.
   * Ideal for formal industrial proposals and stakeholder presentations.

2. **CSV Export**:
   * Exports raw numeric data to a Comma-Separated Values (CSV) format.
   * Ideal for data portability, allowing engineers to perform further customized analysis in Excel, MATLAB, or other data processing tools.
